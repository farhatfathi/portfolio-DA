Automated Machine Learning Pipeline for Customer Churn Prediction

This project integrated GitHub Actions and MLflow. The project began with a raw dataset hosted on GitHub, which was automatically processed using a Python script (automate.py) that mirrored the preprocessing steps initially built in Jupyter Notebook. By leveraging GitHub Actions, every update to the dataset or preprocessing script triggered a continuous integration workflow that executed the pipeline end-to-end. Experiment tracking and model logging were handled using mlflow.autolog(), and the trained model was served locally for real-time inference using MLflow models serve.

#MLOps #Docker #ModelServe #GitAction #Prometheus #Grafana
Image Classification Analysis

Developed an image classification model using a custom CNN and a hybrid MobileNetV2-CNN architecture to classify animal images (cats, dogs, snakes). Applied image augmentation, class balancing, EarlyStopping, and ModelCheckpoint to optimize training. EarlyStopping callback stops training when validation accuracy ≥ 95% for 3 consecutive epochs.
